```mermaid
sequenceDiagram
    actor Κατάστημα
    participant Αρχική σελίδα
    participant Σελίδα ημερολογίου
    participant Σελίδα προσθήκης event
    participant Σελίδα επιλογής χρηστών
    participant Σελίδα απόρριψης
    participant Βάση δεδομένων

    Κατάστημα->>+Αρχική σελίδα: Επιλογή Calendar
    activate Κατάστημα
    Αρχική σελίδα->>+Βάση δεδομένων: Ανάκτηση events
    Βάση δεδομένων-->>Αρχική σελίδα: Events
    alt Αποτυχία
        Βάση δεδομένων-->>-Αρχική σελίδα: Έλειψη δεδομένων
        Αρχική σελίδα->>+Σελίδα απόρριψης: Εμφανίζει
        Κατάστημα->>Σελίδα απόρριψης: Επιλογή επιστροφή
        Σελίδα απόρριψης->>-Σελίδα ημερολογίου: Εμφανίζει
        activate Σελίδα ημερολογίου
    end
    Αρχική σελίδα->>-Σελίδα ημερολογίου: Εμφανίζει

    Κατάστημα->>Σελίδα ημερολογίου: Επιλογή προσθήκη event
    Σελίδα ημερολογίου->>-Σελίδα προσθήκης event: Εμφανίζει
    activate Σελίδα προσθήκης event
    Κατάστημα->>Σελίδα προσθήκης event: Συμπλήρωση στοιχείων event
    Σελίδα προσθήκης event->>+Βάση δεδομένων: Ανάκτηση χρηστών

    Βάση δεδομένων-->>Σελίδα προσθήκης event: Χρήστες
    alt Αποτυχία
        Βάση δεδομένων-->>-Σελίδα προσθήκης event: Έλειψη δεδομένων
        Σελίδα προσθήκης event->>+Σελίδα απόρριψης: Εμφανίζει
        Κατάστημα->>Σελίδα απόρριψης: Επιλογή επιστροφή
        Σελίδα απόρριψης->>-Σελίδα προσθήκης event: Εμφανίζει
    end

    Σελίδα προσθήκης event->>-Σελίδα επιλογής χρηστών: Εμφανίζει
    activate Σελίδα επιλογής χρηστών
    Κατάστημα->>Σελίδα επιλογής χρηστών: Επιλογή χρηστών
    Σελίδα επιλογής χρηστών->>+Βάση δεδομένων: Δημιουργία event
    Βάση δεδομένων-->>-Σελίδα επιλογής χρηστών: Επιτυχία
    Σελίδα επιλογής χρηστών->>-Σελίδα ημερολογίου: Επιστροφή

    deactivate Κατάστημα
```
